SeqMeta

Here you can keep track of your sequencing metadata and format it to for upload to databases such as: GISAID, NCBI and ENA.

Getting started:

If you are new to the application, we recommend that you read trough the workflows first, to familiarize with the app.

Quick links:

  • Metagen-FastQC-Docker, a containerized version of Metagen-FastQC to remove human identifiable reads from raw reads data.
  • Nextflow a platform which enables scalable and reproducible scientific workflows using software containers. It is necessary to have for running nf-core/viralrecon.
  • nf-core/viralrecon v2.1 a bionformatics pipeline to perform assembly and intra-host/low-frequency variant calling for viral samples.
