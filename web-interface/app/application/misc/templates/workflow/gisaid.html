Uploading to GISAID

Download the `.zip` file as described in the guide. This zip file contains the following files:

  • samples.xlsx, this file contains the metadata of the samples
  • fasta files, these are the assembly files.

Submitting the files to GISAID:

  1. Download the latest template file from GISAID, and copy the content of the samples.xlsx into this file. This step was designed so that the user, always checks if the fields of the generated file match those of the GISAID template.
  2. Change the submitter fields in the Excel file to your GISAID username.
  3. Upload the excel file and the assembly files to GISAID using the batch upload option.
