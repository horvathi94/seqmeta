Uploading to NCBI

Download the `.zip` file as described in the guide. This zip file contains the following files:

  • samples.xlsx, this file contains the metadata of the samples.
  • sra.xlsx, this file contains the metadata of the sequencing reads.
  • reads directorythis directory contains the raw reads data.

Submitting files to NCBI:

(It is assumed that you already have an NCBI account.)

  1. Start the upload SRA process.
  2. Create the studies and authors list.
  3. At step number 5 (Biosample Attributes), download the Excel template and copy in the data from the samples.xlsx file.
  4. In the next step (no 6., SRA metadata), download the Excel template and copy in the data from the sra.xlsx file.
  5. Upload the modified Excel template files and upload the reads files from the reads directory.
